    • WealthWise: AI-Powered Financial Planning Web Application

    Enhancing Financial Advisory Through Generative AI, Personalisation, and User-Centric Design

    Developed for BC3415 AI in Accounting & Finance | NTU, 2024 |View Live App

    Project Overview

    Objective: Develop a full-stack web application to enhance the effectiveness of financial advisory services and empower users with tools for informed financial decisions.


    Technology Stack: Flask (Python), HTML, CSS, JavaScript, Google Gemini API, TextBlob, Markdown, Render, EmailJS


    Development Phases:

    • UI Prototyping with Figma
    • Frontend Development in HTML/CSS/JS
    • Backend Development in Flask
    • AI Integration using Gemini API
    • Deployment via Render

    Core Features

    • Expense Tracker: Fast and intuitive interface to record and monitor income and expenses
    • AI Financial Guidance: Personalized saving strategies based on user inputs, powered by Google Gemini
    • Gamified Character Generator: Engage users by visualizing financial personas (simulated for MVP)
    • Email Contact Integration: User queries routed to team inbox via EmailJS

    User Journey & Impact

    WealthWise was designed to support Financial Advisors (FAs) by reducing time spent on routine questions and increasing client engagement. For end-users, the app provides 24/7 personalized guidance and empowers them to take charge of their financial habits through AI-powered insights.

    Reflections & Future Work

    • Ethical Considerations: Incorporate responsible AI practices like transparency and fairness
    • Data Security: Improve data encryption and introduce secure authentication
    • Advanced Features: Add predictive analytics, real-time market integration, and long-term financial goal planning
